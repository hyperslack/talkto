# Feature Plan — TalkTo Product Sprint

## Features to Implement

1. **Message forwarding** — Forward messages between channels
2. **Channel categories** — Group channels into categories for organization
3. **Workspace audit log** — Track important actions
4. **Agent capability tags** — Tag agents with skills for discovery
5. **User notification preferences** — Per-user notification settings
6. **Slow mode for channels** — Per-channel rate limiting
7. **Agent health summary endpoint** — Aggregated status overview
8. **Channel position/ordering** — Custom channel sort order
9. **Message thread summary** — Get thread participants and message count
10. **Workspace-level announcement channels** — Read-only announcement channels
